## Detailed Version (for portfolio/GitHub README)

### FleetPulse - Comprehensive Fleet Management Platform

**Tech Stack:** React 19 | Node.js | Express.js | MongoDB | JWT Authentication | Tailwind CSS | Vite

**Description:**
FleetPulse is a production-ready full-stack web application designed for efficient fleet vehicle management. The platform enables businesses and individuals to track, manage, and analyze their vehicle fleet with comprehensive logging and analytics capabilities.

**Key Features:**
- **Vehicle Management**: Complete CRUD operations with detailed vehicle information (fuel type, registration, transmission, etc.)
- **Fuel Analytics**: Track fuel consumption, costs, and calculate mileage with automated efficiency metrics
- **Maintenance Tracking**: Record maintenance history with configurable reminders (date-based or odometer-based)
- **Insurance Management**: Manage insurance policies with renewal tracking and automated notifications
- **Analytics Dashboard**: Real-time summaries with fuel efficiency calculations, cost analysis, and customizable date filters
- **Smart Reminders**: Dual-trigger reminder system for maintenance and insurance renewals
- **Secure Authentication**: JWT-based auth with refresh token mechanism and HTTP-only cookie storage
- **Advanced Search**: Vehicle search functionality with favorite vehicle management

**Technical Highlights:**
- Implemented dual-token authentication system for enhanced security
- Built hybrid data architecture (MongoDB for persistent data, localStorage for client-side logs)
- Developed automated fuel mileage calculator with cost-per-kilometer analysis
- Created reusable component library for consistent UI/UX
- Designed responsive layouts with mobile-first approach
- Implemented protected routes and comprehensive error handling

**Architecture:**
- Frontend: React with functional components, React Router for navigation, Tailwind CSS for styling
- Backend: RESTful Express.js API with MongoDB/Mongoose for data persistence
- Security: JWT tokens, bcrypt password hashing, CORS configuration, HTTPS implementation
- State Management: React hooks for efficient local state management
